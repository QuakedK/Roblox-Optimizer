# Roblox-Optimizer
Roblox Optimizer's goal is to automate the irritating process of manually optimizing Roblox. This simple and user friendly bat will disable Fullscreen Optimizations, Set the Graphics Preference to high, Set Roblox to High Priority, and Uncap Your FPS!

# Usage
1. Download [Roblox Optimizer](https://github.com/QuakedK/Roblox-Optimizer/releases/download/RobloxFpsUncapper/Roblox-Optimizer-V2.1.bat).
2. Right-click & run it as admin, and almost everything else is automatic, except user-required inputs.
